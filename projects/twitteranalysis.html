Maze Simulation

  • Tech Stack: Python, numpy
  • Github URL: Project Link

For my first year's final assignment in the unit "Fundamentals of Programming," I designed a Maze Simulation using pygame, a project that secured me an impressive grade of 89%. I approached this project with several extensions in mind.

Key requirements included crafting diverse, autonomous movement patterns for the maze entities, and programming them to interact with each other as well as identify and navigate different types of terrain and boundaries. The simulation also featured non-moving targets that entities could either seek or avoid, demonstrating my ability to incorporate target acquisition and pathfinding algorithms. A core component was the presentation of results through a clear and intuitive visual interface, alongside the ability to save this data for further analysis. I also developed a user-friendly game interface using Pygame, pushing my skills in interactive interface design. Extra credit was given for added features and complexity, challenging my creativity and problem-solving skills. Finally, I was required to interpret and discuss the results of various simulation scenarios, honing my analytical abilities. This assignment offered a comprehensive platform to apply and demonstrate the programming skills I had acquired.
